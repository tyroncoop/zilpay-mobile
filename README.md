## Getting Started
Build deploy the local version, from source code.

Clone:
```bash
$ git clone https://github.com/zilpay/zilpay-mobile.git
$ cd zilpay-mobile
```

Install dependencies:
```bash
$ yarn
```

Run IOS:
```bash
$ yarn ios
```

Run real device:
```bash
$ yarn ios --device
$ yarn android --device
```

building release.
```bash
$ yarn ios --variant=release
$ yarn android --variant=release
```

build bungle
```bash
node node_modules/react-native/local-cli/cli.js bundle --entry-file='./index.js' --bundle-output='./ios/main.jsbundle' --dev=false --platform='ios' --assets-dest='./ios'
```

Android build singed package
```bash
 $ ./gradlew bundleRelease
```

### Nodeify

If you've an issue regarding crypto library when running the app. Run this command:
`node_modules/.bin/rn-nodeify --install "assert,process,buffer,crypto,stream,events,constants" --yarn --hack`
